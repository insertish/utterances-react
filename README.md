# utterances-react

This is React component for [utterances](https://utteranc.es/)

## Install

```bash
# npm
$ npm i utterances-react
# yarn
$ yarn add utterances-react
```

## Usage

```js
import Utterances from "utterances-react"

export default ({ data }) => {
    return (
        <Utterances repo="wicksome/utterances-react"/>
    )
}
```
